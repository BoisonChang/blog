* **選擇排序法（Selection Sort）**

  * 找到最小值，移到最左邊

* **泡沫排序法（Bubble Sort）**

  * 跟隔壁互相比較，順序錯了就交換，讓大的元素一直浮到最後

* **插入排序法（Insertion Sort）**

  * 一個一個照順序插入到順序對的位置

* **合併排序法（Merge Sort）**

  * 切一半，排好左邊排好右邊再合併

## 一、選擇排序法（Selection Sort）

* 時間複雜度: O(n^2)

```javascript
const selectionSort = (arr) => {
  const length = arr.length;
  for (let i = 0; i < length; i++) {
    let min = arr[i];
    let minIndex = i;
    for (let j = i; j < length; j++) {
      if (arr[j] < min) {
        min = arr[j];
        minIndex = j;
      }
    }  
    // ES6 的用法，交換兩個數值
    [arr[minIndex], arr[i]] = [arr[i], arr[minIndex]];
  }
  return arr;
}

let arr = [1,3,5,7,2,4,8,9]
selectionSort(arr) //  [1, 2, 3, 4, 5, 7, 8, 9]
```

## 二、泡沫排序法

> 稱為「泡沫」排序法，因為元素很像「浮」了上來

* 平均時間複雜度: O(n^2)

* 最壞時間複雜度: O(n^2)

* 最優時間複雜度: O(n) 

加上 swapped，如果輸入是已經排好的陣列就只會跑一次內圈，然後就跳掉了，單純跑完一輪不改動，所以時間複雜度會是`O(n)`。

```javascript
function bubbleSort = (arr) => {
  const  n = arr.length;
  let swapped = true;  
  for (let i = 0; i < n && swapped; i++) {
    swapped = false;
    for (let j = 0; j < n - 1 - i; j++) {
      if (arr[j] > arr[j + 1]) {
        swapped = true;
        [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
      }
    }
  }
  return arr;
}

let arr = [1,3,5,7,2,4,8,9]
selectionSort(arr) //  [1, 2, 3, 4, 5, 7, 8, 9]
```
